# GreenLoop HKS Worker Training Guide (Malayalam)

## നന്ദി (Thank You) - GreenLoop-ലേക്ക് സ്വാഗതം!
GreenLoop പ്ലാറ്റ്‌ഫോം ഉപയോഗിച്ച് മാലിന്യ സംസ്കരണം കൂടുതൽ എളുപ്പമാക്കാം.

### 1. ലോഗിൻ ചെയ്യുക (Login)
- നിങ്ങളുടെ രജിസ്റ്റർ ചെയ്ത ഫോൺ നമ്പറും പാസ്‌വേഡും ഉപയോഗിച്ച് ലോഗിൻ ചെയ്യുക.
- നിങ്ങളുടെ വാർഡ് (Ward) ഹോം സ്ക്രീനിൽ കാണാൻ സാധിക്കും.

### 2. പിക്കപ്പ് വിവരങ്ങൾ (Pickup Details)
- ഓരോ ദിവസത്തെയും പിക്കപ്പുകൾ 'Tasks' സെക്ഷനിൽ കാണാം.
- പുതിയ പിക്കപ്പുകൾ വരുമ്പോൾ നിങ്ങൾക്ക് നോട്ടിഫിക്കേഷൻ ലഭിക്കും.

### 3. ക്യുആർ കോഡ് വെരിഫിക്കേഷൻ (QR Code Verification)
- ഓരോ വീട്ടിൽ നിന്നും മാലിന്യം ശേഖരിക്കുമ്പോൾ ആ വീട്ടിലെ QR കോഡ് സ്കാൻ ചെയ്യുക.
- മാലിന്യത്തിന്റെ അളവ് (Weight) രേഖപ്പെടുത്തുക.

### 4. പോയിന്റുകൾ നൽകൽ (Awarding Points)
- പിക്കപ്പ് പൂർത്തിയാക്കുമ്പോൾ റെസിഡന്റിന് ഗ്രീൻലീഫ് (Greenleaf) പോയിന്റുകൾ ലഭിക്കും.
